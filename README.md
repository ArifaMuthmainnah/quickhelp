# QuickHelp
## Anggota Kelompok
* Arifa Muthmainnah (2308107010012)
* Sifa Jema (2308107010080)

  
**QuickHelp** adalah aplikasi mobile berbasis Flutter yang dirancang untuk membantu mahasiswa dalam mencari maupun memberikan bantuan secara cepat di lingkungan kampus.
Aplikasi ini memungkinkan pengguna membuat permintaan bantuan berdasarkan kategori tertentu, menampilkan lokasi bantuan, serta berkomunikasi dengan pengguna lain melalui fitur chat. Dengan memanfaatkan teknologi berbasis lokasi (*Location Based Service*), mahasiswa dapat menemukan bantuan yang tersedia di sekitar mereka dengan lebih mudah dan efisien.

## Kontribusi terhadap Sustainable Development Goals (SDGs)

QuickHelp mendukung beberapa tujuan Sustainable Development Goals (SDGs), khususnya:

### SDG 4 – Quality Education (Pendidikan Berkualitas)
QuickHelp membantu mahasiswa memperoleh dukungan akademik dengan lebih mudah, seperti mencari teman diskusi, berbagi informasi, meminjam perangkat belajar, dan memperoleh bantuan yang mendukung proses pembelajaran.

### SDG 11 – Sustainable Cities and Communities (Kota dan Komunitas Berkelanjutan)
QuickHelp mendorong terbentuknya komunitas kampus yang saling membantu, inklusif, dan kolaboratif melalui sistem berbagi bantuan berbasis lokasi sehingga memperkuat hubungan sosial antar mahasiswa.

## Cara Menjalankan Project

Pastikan telah menginstal:

* Flutter SDK
* Android Studio
* Android SDK
* Git

Cek instalasi Flutter:

```bash
flutter doctor
```

### Clone Repository

```bash
git clone https://github.com/ArifaMuthmainnah/quickhelp.git
```

Masuk ke folder project:

```bash
cd quickhelp
```

### Install Dependencies

```bash
flutter pub get
```

### Menjalankan Emulator

1. Buka Android Studio
2. Pilih **Device Manager**
3. Jalankan Android Emulator
4. Pastikan emulator aktif

### Menjalankan Aplikasi

```bash
flutter run
```
Atau tekan tombol **Run** pada Android Studio.
